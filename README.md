Visiting card website for a client with physiotheraphy practice.
It's supposed to be a simple in navigation page to check basic information about client's practice. </br>
Target group are adults and elderly people. </br>
Page is made with use of minimalistic color palette to look professional and be intuitive in use.
The form is single-page design to give option between using the navigation bar and simply scrolling down the page.
I used Ubuntu-sans font because of it's simple and easy to read style.
